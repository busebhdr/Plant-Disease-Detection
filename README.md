# 🌿 Plant Disease Detection

Deep learning project for automatic plant disease detection from leaf images.

## Dataset
- **Source:** [New Plant Diseases Dataset (Kaggle)](https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset)
- **Images:** 87,000+ leaf images
- **Classes:** 38 (14 plant species)
- **Split:** 70,295 train / 17,572 validation

## Models

### Model 1 — Custom CNN
- **Validation Accuracy: 97.98%**

### Model 2 — EfficientNetB0 (Transfer Learning)
- **Validation Accuracy: 99.19%**

## Technologies
- Python, TensorFlow, Keras
- Google Colab
- scikit-learn, Matplotlib, Seaborn
